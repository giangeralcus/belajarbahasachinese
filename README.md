# Learn Chinese Beginner

Public repo buat journey belajar Mandarin dari nol.

## Goal
- Build daily habit (20-30 menit/hari)
- Pinyin + tones foundation
- Basic conversation and survival vocabulary

## Study Plan (First 4 Weeks)
1. Week 1: Pinyin, initials/finals, 4 tones, numbers, greetings
2. Week 2: Pronouns, basic sentence patterns, family words
3. Week 3: Daily activities, time/date, simple questions
4. Week 4: Review + short self-introduction recording

## Repo Structure
- `notes/`: ringkasan materi harian
- `vocab/`: daftar kosakata by topic
- `practice/`: latihan speaking/listening/hanzi
- `resources/`: links ke apps, channels, references

## How to Use
1. Commit progress harian (even small).
2. Keep vocab list simple and consistent.
3. Add weekly reflection in `notes/`.
